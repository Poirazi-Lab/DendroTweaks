Units
=====

DendroTweaks uses the same unit conventions as the `NEURON <https://www.neuron.yale.edu/neuron/static/docs/units/unitchart.html>`_ simulator.

+---------------+---------------+--------------------------------+
| Category      | Variable      | Units                          |
+===============+===============+================================+
| Time          | t             | [ms]                           |
+---------------+---------------+--------------------------------+
| Voltage       | v             | [mV]                           |
+---------------+---------------+--------------------------------+
| Current       | i             | [mA/cm2] (distributed)         |
|               |               | [nA] (point process)           |
+---------------+---------------+--------------------------------+
| Concentration | ko, ki, etc.  | [mM]                           |
+---------------+---------------+--------------------------------+
| Specific      | cm            | [uf/cm2]                       |
| capacitance   |               |                                |
+---------------+---------------+--------------------------------+
| Length        | diam, L       | [um]                           |
+---------------+---------------+--------------------------------+
| Conductance   |               | [S/cm2] (distributed)          |
|               |               | [uS] (point process)           |
+---------------+---------------+--------------------------------+
| Cytoplasmic   | Ra            | [ohm cm]                       |
| resistivity   |               |                                |
+---------------+---------------+--------------------------------+
| Resistance    | Ri            | [106 ohm]                      |
+---------------+---------------+--------------------------------+
